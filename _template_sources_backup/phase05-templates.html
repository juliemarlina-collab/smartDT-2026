T14 User Feedback Form
Record feedback from real target users during prototype testing.
Not Started
A1
Interactive Activity
Select testers: use at least 3 real target users who match your persona.
Observe silently: do not explain too much or defend the design.
Capture: write confusion moments, direct quotes and ratings.
Sample Feedback Entry

Task: Try to pre-order lunch and select a pickup time.

Observation: Tester understood the menu but hesitated at pickup time selection.

Quote: “I need confirmation that my order is successful.”

Testing Rule

Watching users struggle is useful data. Stay calm and observe before helping.
